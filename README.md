# RenderingEngine

A rendering engine in C++ with OpenGL that I intend to progressively improve.

## How to use

This project runs with CMake.
1 - Clone the repository
2 - Reload the CMake project

GLM and GLFW should be fetched automatically.

## Project goals
[ ] - Showing a triangle using a simple shader

[ ] - Showing some primitive shapes

[ ] - Having a 3D camera with perspective

[ ] - Implementing a Phong Lambert shader

[ ] - Fully controllable camera

[ ] - Loading FBX or OBJ

[ ] - Supporting UVs and textures

[ ] - Implementing a UI inside of the engine

[ ] - Working scene architecture
